# MyBlogwebsite 

MyBlogwebsite is a simple blog web application built with Node.js, Express.js, and EJS.  
It allows users to create, view, edit, and delete blog posts, demonstrating dynamic content rendering and basic backend logic.

# Overview
This project simulates a basic blogging platform where users can manage their own blogs.  
Blog posts are stored using an in-memory array, and pages are rendered dynamically using EJS templates. Itâ€™s a beginner-friendly full-stack project showing how server-side logic interacts with frontend views.

- Create new blog posts  
-  View list of all blog posts  
-  Edit existing blog posts  
-  Delete blog posts  
-  Server-side rendering with EJS
